# Violin Edu

Mobile-first violin fingering practice web app built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, and Tone.js.

## Features

- Interactive violin fingerboard for G, D, A, and E strings
- Solfege, quiz, and song practice modes
- Chromatic fingering toggle for L1, L2, and H3 positions
- Tap-to-hear pitch playback with optional haptics
- PWA manifest and service worker assets
- Reference learning images from `01 Source`

## Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Verification

```bash
npm test
npm run build
```
